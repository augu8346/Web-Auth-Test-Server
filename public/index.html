Auth Test Server

A self-contained testing environment for validating HTTP authentication flows with GDAL, ArcObjects, and other clients.

🚀 Open Test Client 📡 View Live Logs

No Auth / Custom

Test custom headers and cookies without mandatory authentication.

  • Endpoint: /test/none/cog.tif

Basic Auth

Test standard HTTP Basic Authentication (also used for netrc testing).

  • Endpoint: /test/basic/cog.tif
  • User: admin
  • Password: admin

Bearer Token

Test Authorization Bearer token workflows.

  • Endpoint: /test/bearer/cog.tif
  • Token: supersecrettoken123

OAuth 2.0

Simulate OAuth workflows (Auth Code / Client Credentials).

  • Auth: /oauth/authorize
  • Token: /oauth/token
  • Target: /test/oauth2/cog.tif

mTLS (Client Certs)

Test Mutual TLS using PKI or PKCS #12 certificates.

  • PKI (PEM): /test/mtls-pki/cog.tif
  • PKCS#12: /test/mtls-pkcs12/cog.tif
  • Certs located in the certs/ folder.
